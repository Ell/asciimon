Twitch Plays Pokemon is a stream where thousands of people try to collaboratively complete Pokemon by providing button input on chat. This site takes that stream, runs it through character recognition, and displays it live over websockets using Python, OpenCV, Flask, and a little bit of redis. All of the code is opensource and can be found on GitHub at ell/asciimon. Most of the interesting stuff is found in the rmmh/pokr repository.

Hosting is 50% of a $5 DigitalOcean Box's CPU and 250GB/mo bandwidth to pull down the twitch stream. Not too bad in my opinion.
